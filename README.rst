Prerequisite
----------------------------------------

Install dependencies

.. code-block:: bash

    $ pip install fasttext


Installation
----------------------------------------


Stable version

.. code-block:: bash

    $ pip install https://github.com/undertheseanlp/languageflow/archive/master.zip

Latest version

.. code-block:: bash

    $ pip install https://github.com/undertheseanlp/languageflow/archive/develop.zip
